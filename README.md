# Web blog de salud y salud mental

En esta web se publicarán distintas paginas con articulos sobre distintas áreas de la salud, sobre todo se centrará en la salud mental, también, se creará un servidor de discord donde se publicarán los cambios de la web, anuncios importantes, charlas, salas donde hablar sobre la salud y compartir conocimientos y habrá salas especiales privadas a modo de "confesionario" donde las personas se podrán desahogar, pedir ayuda y ser ayudados por otras personas.
